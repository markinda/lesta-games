# lesta-games-test-work
#### _Install_
```shell script
yarn install
```
# Usage
### Development server

```shell script
yarn start
```

You can view the development server at: http://localhost:3000/

You can view the bundle Analyzer server at: http://192.168.1.38:3000/

### Production build

```shell script
yarn build
```

You can view build in ../build folder
